TechDivision_StompProtocol (WIP)
==========================


### Installation 

* Clone repository or Download the ZIP
* Add the  the followings lines to /opt/appserver/composer.json

```json
    "repositories": [
        {
            "type":"vcs",
            "url":"/YOUR/PATH/stomp"
        }
    ],
    "require": {
    "appserver-io/stomp" : "dev-master"
    },
```
